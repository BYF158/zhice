## 1. 引言

### 1.1 编写目的

本文档详细描述智测心理测试系统的数据库结构设计，包括表结构、字段定义、索引设计、外键关系等，为数据库实现和维护提供依据。

### 1.2 背景

本系统采用MySQL 8.0作为数据库，需要设计合理的数据库结构以支持系统功能需求。

### 1.3 定义

| 术语 | 定义 |
|-----|------|
| 主键 | 唯一标识表中每一条记录的键 |
| 外键 | 关联到其他表主键的键 |
| 索引 | 提高查询效率的数据结构 |

## 3. 数据库命名规范

### 3.1 表命名

- 使用小写字母
- 使用下划线分隔单词
- 使用复数形式
- 示例：`personality_results`, `personality_answers`

### 3.2 字段命名

- 使用小写字母
- 使用下划线分隔单词
- 示例：`user_id`, `test_date`

### 3.3 索引命名

- 普通索引：`idx_字段名`
- 唯一索引：`uk_字段名`
- 外键索引：`fk_字段名`

## 8. 数据备份策略

### 8.1 备份计划

- **全量备份**：每日凌晨2点
- **增量备份**：每4小时
- **保留策略**：保留最近30天

### 8.2 备份脚本

```bash
#!/bin/bash
# 数据库备份脚本

DATE=$(date +%Y%m%d_%H%M%S)
BACKUP_DIR="/data/backup/mysql"
DB_NAME="ry-vue"
DB_USER="root"
DB_PASS="password"

# 创建备份目录
mkdir -p $BACKUP_DIR

# 执行备份
mysqldump -u$DB_USER -p$DB_PASS $DB_NAME | gzip > $BACKUP_DIR/$DB_NAME_$DATE.sql.gz

# 删除30天前的备份
find $BACKUP_DIR -name "*.sql.gz" -mtime +30 -exec rm -f {} \;
```
