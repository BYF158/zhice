# 智测心理测试系统使用手册

## 系统简介

智测心理测试系统（Zhice）是基于若依（RuoYi）v3.9.0框架开发的心理测试平台，专注于人格类型测评。系统采用前后端分离架构，前端使用Vue + Element UI，后端采用Spring Boot + Spring Security，为用户提供专业、便捷的心理测评服务。

## 系统功能

### 1. 人格测试

用户可以通过系统完成人格类型测试，了解自己的性格倾向。

**测试流程：**
1. 登录系统
2. 进入人格测试模块
3. 逐题作答，每道题选择最符合自己的选项
4. 提交测试结果
5. 查看个性化分析报告

**测试特点：**
- 包含多项选择题，每题有多个选项
- 测试采用分页展示，避免一次性加载过多题目
- 自动记录用户的答题记录
- 提供详细的人格分析报告

### 2. 个人中心

用户可以在个人中心查看自己的测试历史记录，包括：
- 测试时间
- 测试类型
- 测试结果
- 人格分析

### 3. 结果查询

系统支持查询最近一次的测试结果，包括四个维度的得分：
- C类型得分
- P类型得分
- M类型得分
- S类型得分

## 用户操作指南

### 第一次使用

1. **账号注册/登录**
   - 使用管理员提供的账号登录系统
   - 默认管理员账号：`admin`
   - 默认密码：`admin123`（首次登录后建议修改）

2. **进入测试**
   - 登录后，在导航栏找到"人格测试"模块
   - 点击进入测试页面

3. **完成测试**
   - 按顺序回答每一道题目
   - 每题选择一个最符合自己的选项
   - 完成所有题目后提交

4. **查看结果**
   - 提交后系统自动计算分数
   - 点击"查看结果"按钮查看详细报告

### 测试历史查看

1. 进入"个人中心"
2. 点击"测试记录"
3. 可以查看所有历史测试记录
4. 点击某次记录可以查看详细信息

## 管理员功能

### 题目管理

管理员可以对测试题目进行管理，包括：

**查看题目列表**
- 进入"题目管理"模块
- 系统会显示所有题目的列表
- 支持按题目类别筛选

**新增题目**
1. 点击"新增"按钮
2. 填写题干内容
3. 选择题目类别
4. 保存

**编辑题目**
1. 在题目列表中找到要编辑的题目
2. 点击"编辑"按钮
3. 修改题目内容
4. 保存

**删除题目**
1. 勾选要删除的题目
2. 点击"删除"按钮
3. 确认删除

### 用户管理

管理员可以管理系统用户：

**查看用户列表**
- 进入"用户管理"模块
- 系统显示所有注册用户

**新增用户**
1. 点击"新增"按钮
2. 填写用户信息
3. 设置用户角色
4. 保存

**编辑用户**
- 修改用户基本信息
- 调整用户角色和权限
- 重置用户密码

**删除用户**
- 删除不需要的用户账号
- 删除前会提示确认

### 测试记录管理

管理员可以查看所有用户的测试记录：

**查看记录列表**
- 进入"测试记录"模块
- 系统显示所有用户的测试记录
- 可以按用户、时间筛选

**查看详细记录**
- 点击某条记录查看详情
- 包括用户信息和答题详情

### 结果管理

管理员可以查看和管理测试结果：

**查看结果列表**
- 进入"结果管理"模块
- 查看所有用户的测试结果
- 可以按用户、结果类型筛选

**导出结果**
- 支持将测试结果导出为Excel文件
- 便于进行数据分析和报告生成

## 常见问题

### Q1: 测试中途退出怎么办？

A: 系统会自动保存已答题目，下次登录可以继续完成。

### Q2: 可以重复测试吗？

A: 可以。用户可以多次参加测试，系统会记录每次测试的结果。

### Q3: 测试结果准确吗？

A: 测试结果基于科学的评分算法，但仅供参考。如需专业心理评估，请咨询专业心理咨询师。

### Q4: 忘记密码怎么办？

A: 可以联系管理员重置密码，或使用密码找回功能（如果已开启）。

### Q5: 如何修改个人信息？

A: 登录后进入个人中心，可以修改基本信息和密码。

## 注意事项

1. **隐私保护**
   - 用户的测试数据严格保密
   - 不会将个人信息透露给第三方
   - 测试结果仅用于个人参考

2. **测试建议**
   - 在安静的环境下完成测试
   - 保持放松的心态
   - 诚实回答每个问题
   - 不要过度思考，以第一直觉为准

3. **结果解读**
   - 测试结果仅供参考
   - 人格是复杂和多维的
   - 不要过于依赖测试结果做重要决定

4. **使用规范**
   - 不得用于商业用途
   - 不得用于歧视性评价
   - 尊重个人隐私和数据安全
