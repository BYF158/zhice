## 核心业务流程

### 1. 人格测试流程

```
用户登录 → 选择测试 → 加载题目（分页）→ 答题 → 提交答案 → 
保存记录 → 计算分数 → 保存结果 → 显示报告
```

**关键代码：**

```java
// 提交测试
@PostMapping("/add")
public R<?> submitPersonalityTest(@RequestBody PersonalityTestRequest request) {
    // 1. 创建测试记录
    Long recordId = personalityRecordService.saveRecord(userId);
    
    // 2. 保存答案
    for (Integer optionId : selectedOptionIds) {
        PersonalityAnswer answer = new PersonalityAnswer();
        answer.setRecordId(recordId);
        answer.setOptionId(optionId);
        answers.add(answer);
    }
    personalityAnswerService.saveAll(answers);
    
    return R.ok(recordId, "提交成功");
}
```

### 2. 评分算法

系统根据用户选择的选项计算各维度的得分：

```java
// 获取评分
PersonalityScoreResponse getPersonalityScores(Integer recordId) {
    // 1. 查询用户答案
    List<PersonalityAnswer> answers = personalityAnswerService.findByRecordId(recordId);
    
    // 2. 查询选项对应的分数
    Map<Long, Integer> optionScores = getOptionScores(answers);
    
    // 3. 按维度汇总分数
    int typeCScore = calculateTypeC(optionScores);
    int typePScore = calculateTypeP(optionScores);
    int typeMScore = calculateTypeM(optionScores);
    int typeSScore = calculateTypeS(optionScores);
    
    // 4. 返回结果
    return new PersonalityScoreResponse(typeCScore, typePScore, typeMScore, typeSScore);
}
```

### 3. 结果保存流程

```java
@PostMapping("/add")
public R<?> submitPersonalityTest(@RequestBody PersonalityResults results) {
    // 1. 查询最新测试时间
    Date testDate = personalityRecordService.getLatestTestTimeByUserId(userId);
    
    // 2. 保存结果
    PersonalityResult result = new PersonalityResult();
    result.setUserId(userId);
    result.setTestDate(testDate);
    result.setTypeCScore(typeCScore);
    result.setTypePScore(typePScore);
    result.setTypeMScore(typeMScore);
    result.setTypeSScore(typeSScore);
    personalityResultService.save(result);
    
    return R.ok(result.getResultId(), "保存成功");
}
```

## 常见问题

### 1. 数据库连接失败

**原因：** 数据库配置错误或数据库未启动

**解决：**
- 检查 `application-druid.yml` 中的数据库配置
- 确保MySQL服务已启动
- 检查数据库用户名和密码是否正确

### 2. 前端请求跨域

**原因：** 前端和后端端口不同

**解决：**
在 `zhice-framework` 中配置CORS：
```java
@Configuration
public class ResourcesConfig implements WebMvcConfigurer {
    @Override
    public void addCorsMappings(CorsRegistry registry) {
        registry.addMapping("/**")
                .allowedOriginPatterns("*")
                .allowCredentials(true)
                .allowedMethods("GET", "POST", "PUT", "DELETE")
                .maxAge(3600);
    }
}
```

### 3. JWT Token过期

**原因：** Token有效时间已过

**解决：**
- 重新登录获取新Token
- 或修改 `application.yml` 中的Token有效期配置
